# 🎲 Dice with Death!

> **주사위를 굴려라, 운명을 결정하라!**  
> 턴제 텍스트 RPG — Visual Studio C++ 프로젝트


## 🎮 게임 소개

**Dice with Death!** 는 주사위를 굴려 나온 면으로 플레이어의 행동을 결정하는 **턴제 텍스트 RPG**입니다.

- 매 턴 주사위를 굴려 행동이 **확률적**으로 결정됩니다
- 3인 파티를 구성하고 던전을 탐험합니다
- 골드와 경험치로 파티를 강화하며 최종 보스에 도전합니다
- 전멸해도 괜찮습니다 — **이전 파티의 골드는 다음 파티가 이어받습니다!**


## ✨ 주요 특징

- 🎲 **주사위 기반 액션 시스템** — 매 턴 주사위를 굴려 행동이 결정됩니다
- ⚔️ **클래스 선택 & 파티 구성** — 3개의 클래스를 조합해 나만의 파티를 꾸립니다

## 🎮 게임 플레이

```
1. 게임 시작 — 클래스를 선택해 3인 파티를 구성합니다
2. 던전 입장 — 탐험을 시작합니다
3. 턴 진행  — 주사위를 굴려 행동을 결정합니다
4. 전투 / 이벤트 — 결과에 따라 다양한 상황이 펼쳐집니다
5. 던전 클리어 또는 전멸 — 살아남을 수 있을까요?
```

## 🧙 클래스 & 주사위

파티는 총 **3명**으로 구성되며, 아래 5개 클래스 중 자유롭게 조합할 수 있습니다.

| 클래스 | 특징 |
|--------|------|
| ⚔️ **Fighter** | 공격 위주의 주사위 |
| 🏹 **Archer** | Fighter보다 강력한 공격을 보유하지만, None 면이 2개 포함된 하이리스크 주사위 |
| 🛡️ **Defender** | 방어 위주의 주사위 |
| 🗡️ **Rogue** | 출혈·스턴을 유발하는 공격 위주 주사위 |
| 💊 **Cleric** | 치유 위주 주사위 |


### 기본 행동

| 행동 | 설명 |
|------|------|
| ⚔️ **Attack** | 대상에게 피해를 입힘. Shield를 먼저 차감 |
| 🛡️ **Defence** | 대상에게 Shield 부여. Shield는 다음 턴에 0이 됨 |
| 💊 **Heal** | 대상의 체력 회복. 최대 체력 초과 불가, 사망한 대상은 회복 불가 |



### 상태이상

| 상태이상 | 효과 |
|----------|------|
| 🩸 **출혈** | 매 턴마다 체력의 10% 감소 |
| 😵 **스턴** | 지정된 턴 수만큼 행동 불가 |

> 💡 동일한 상태이상이 중첩되면 **턴 수가 합산**됩니다.  
> (예: 출혈 2턴 남은 적에게 출혈(3) 적중 → 출혈 **5턴** 지속)



## 🛒 상점 시스템

### 언제 방문하나?
게임 루프 내 **두 번** 방문할 수 있으며, 전투에서 획득한 골드를 사용합니다.

### 무엇을 할 수 있나?

| 메뉴 | 설명 |
|------|------|
| 💤 **휴식** | 50골드를 내고 파티 체력 회복 |
| 🧪 **아이템 구매** | 전투 중 사용 가능한 포션 구입 (최대 3개 보유) |
| 📦 **인벤토리 확인** | 보유 중인 아이템 목록 확인 |
| 🎲 **다이스 면(Action) 구입** | 원하는 면을 구입해 파티원의 주사위를 커스터마이징 |

> 💡 다이스 면 구매 순서: **구입할 면 선택 → 적용할 파티원 결정 → 교체할 면 선택**
